Fictional Character API Documentation

Host: localhost:8080

Resource Directory: /fictionalCharacterAPI_war/service

Resources:

  • /name
  • /insert_character

/name

Method: GET

Required Parameters

  • 1. Id

    • Optional Parameters

  • None

    • Example: localhost:8080/fictionalCharacterAPI_war/service/name/1

    The /name resource uses the GET method and a number parameter to fetch a character in the fictional character database according to its id

    /insert_character

    Method: POST

    Required Parameters

  • 1. Name
  • 2. Media
  • 3. Franchise
  • 4. Species
  • 5. Alignment

    • Optional Parameters

  • None

    • Example: localhost:8080/fictionCharacterAPI_war/service/insert_character

    Send form data to the endpoint to insert a new fictional character into the database
