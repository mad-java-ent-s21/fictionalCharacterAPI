Character Fictional API

Documentation

Host: https://localhost/8080

Rescource directory: /fictionalCharacterAPI/service

Rescources:

  • /name
  • /insert_character

/name

GET

Required Parameters

1. Id

Optional Parameters

None

Example: https://localhost/8080/service/name/1

The /name resource uses the GET method and a number parameter to fetch a character in the fictional character database according to its id

/insert_character

POST

Required Parameters

1. Name

2. Media

3. Franchise

4. Species

5. Alignment

Optional Parameters

None

Example: https://localhost/8080/service/insert_character

Send form data to the endpoint to insert a new fictional character into the database
